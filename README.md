## Setup

```sh
git clone https://github.com/GPT-Engineer-App/emerald-ai-unity3d-meta.git
cd emerald-ai-unity3d-meta
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
